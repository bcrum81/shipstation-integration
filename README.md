<b>Shipstation Integration</b><br/>
Wordpress Plugin<br/>
Fall, 2025<br/>
<br/>
This plugin is designed to give users the ability to share ShipStation order statuses with clients.<br/>
Nativly ShipStation does not have a user permissions set that would allow you to give aceess to certain stores to individual users (clients). This means that any updates, changes or actions a client might want to take on an order that your business is fulfilling needs to happen outside of the system. This is combersome and can lead to dropped communications or missed oportunities.<br/>
With this WordPress plugin you can give access to certain stores to individual clients and allow them to place orders on hold, add notes, updated shipping addresses and various other common actions. Give the power to your clients and ship on!
